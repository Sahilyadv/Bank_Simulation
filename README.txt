# About Project

The **ABC Bank - Banking Simulation System** is a desktop application developed in **Python** using **Tkinter** for the graphical user interface and **MySQL** as the database.

The system simulates real-world banking operations such as account creation, secure login, deposits, withdrawals, money transfers, balance inquiry, password recovery using OTP, and email notifications.

This project is developed for learning database management, GUI programming, and Python application development.

---

- Secure Login System

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/Sahilyadv/Banking-Simulation-System.git
```

## Go to Project

```bash
abcd Banking-Simulation-System
```

## Install Requirements

```bash
pip install -r requirements.txt
```

## Create Database

```sql
CREATE DATABASE banking_system;
```

Import the SQL file into MySQL.

Update database credentials in **database.py**

Run the project

```bash
python main.py
